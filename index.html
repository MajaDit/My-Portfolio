React Weather App

As one of the final projects for SheCodes React, I built a weather application using the JS React library.

Working on this, I learnt how to work with React components, React event listeners, React states, loops and forms, as well as how to call and use an API in a React application. I included a React external component to implement animated icons and used the dependencies Bootstrap and Axios.

The app currently has a unit conversion on today's temperature. In the future, I am planning on adding a conversion to Fahrenheit on the forecast as well.
